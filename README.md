# 🚀 Featured Project

## GST Data Processing System

### Overview

Developed a scalable system for **fetching, validating, and processing GST data** using **AWS S3 integration and multi-threaded processing**.

### Key Features

- Multi-threaded processing using **ExecutorService**
- AWS S3 integration for file management
- JSON parsing using **Jackson**
- GSTIN & PAN validation using Regex
- Duplicate record prevention using **Spring Data JPA**
- Structured logging and exception handling

Java | Spring Boot | AWS S3 | MySQL | ExecutorService | JSON
